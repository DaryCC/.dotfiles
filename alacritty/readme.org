* Instalación desde repositorio de Arch
https://wiki.archlinux.org/title/Alacritty
#+begin_src shell
  git clone https://aur.archlinux.org/alacritty-git.git
  cd alacritty
  makepkg -si
#+end_src
* simbolic link
#+begin_src shell

❯ ln -s ~/.spacemacs.d/alacritty/alacritty.yml ~/.config/alacritty
#+end_src
